# idux-playground

基于`@vue/repl`实现

## 技术栈

- `@vue/repl`
- vue3.x
- vite
- unocss
- typescript
- pnpm
- rollup（使用 rollup 对组件库进行二次打包）

## 快速开始

```js
pnpm i

// 每次更新vue3、idux依赖都要重新build一次，才能run dev
pnpm build

pnpm dev
```
