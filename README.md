# iDux Playground

基于`@vue/repl`实现，直接访问[iDux Playground](https://playground.idux.site/)查看效果

## 技术栈

- `@vue/repl`
- vue3.x
- vite
- unocss
- typescript
- pnpm

## 快速开始

```js
pnpm i

pnpm dev
```
